## 🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

**Решение:** Используйте веб-интерфейс GitHub для загрузки файлов

### Проблема: Ошибка при экспорте в PDF
**Решение:** 
1. Используйте Print → Save as PDF в браузере
2. Или установите: `pip install nbconvert`

### Проблема: awk/sed не работают в Windows
**Решение:** 
1. Обязательно используйте **Git Bash**, не Command Prompt!
2. Или сделайте скриншоты в Jupyter Notebook (там тоже выполняются команды)
