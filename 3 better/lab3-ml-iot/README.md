## Описание лабораторной работы

Лабораторная работа посвящена изучению утилит командной строки `awk` и `sed` для обработки текстовых данных, а также практическому применению библиотек Python для анализа и визуализации данных.

### Основные задачи:
1. Изучение примеров использования awk и sed
2. Освоение основ языка сценариев AWK
3. Работа с потоковым редактором sed
4. Применение awk и sed для решения практических задач
5. Создание приложений с визуализацией данных (Panel App, Voila)
6-7. Анализ табличных данных

## Инструкция по запуску

### Подготовка окружения

1. Установите Anaconda или Miniconda
2. Создайте виртуальное окружение:
```bash
conda create -n ml-iot python=3.8
conda activate ml-iot
```

3. Установите необходимые пакеты:
```bash
pip install jupyter pandas plotly panel voila
```

### Запуск Jupyter Notebook

```bash
cd lab3-ml-iot
jupyter notebook
```

### Выполнение заданий

Каждое задание находится в отдельной папке `src/taskN/`. Откройте соответствующий `.ipynb` файл и выполните все ячейки последовательно.
