### ✅ Задание 4: Вариант 5
**Файл:** `src/task4/lab3-task4.ipynb`

#### Задача 1: Поиск "Network Protocol" в RFC файле
**Решение:**
```awk
awk '/Network Protocol/ {count++; print NR": "$0} 
    END {print "Найдено строк:", count}' rfc-ref.txt
```
- Автоматическая загрузка файла с сервера RFC
- Поиск паттерна
- Подсчет количества
- Вывод с номерами строк

#### Задача 2: Площадь и периметр параллелограмма
**Решение:** Создан AWK скрипт с функциями:
```awk
function area(base, height) {
    return base * height
}

function perimeter(side_a, side_b) {
    return 2 * (side_a + side_b)
}

function main(a, b, h) {
    # вызов функций и вывод результатов
}
```
- Математические формулы реализованы
- Красивый форматированный вывод
- Два примера расчетов

#### Задача 3: Замена пробела на 0 перед цифрой
**Решение:**
```bash
sed 's/^ \([0-9]\)/0\1/' file.txt
```
- Использование регулярных выражений
- Группы захвата \\1
- Создан тестовый файл для демонстрации
