Задание 6 предоставлено в виде HTML файла с готовыми примерами кода.

## 📁 Файл задания

**Файл:** `lab3-6.html`

Этот файл содержит Jupyter Notebook, экспортированный в HTML формат.

## 🚀 Инструкция по выполнению

### Вариант A: Импорт HTML в Jupyter (Рекомендуется)

К сожалению, Jupyter не умеет напрямую импортировать HTML обратно в .ipynb формат.

### Вариант B: Ручное копирование кода

1. **Откройте файл `lab3-6.html` в браузере**
   - Просто двойной клик по файлу

2. **Создайте новый блокнот:**
   ```bash
   jupyter notebook
   ```
   - File → New → Notebook
   - Назовите: `lab3-task6.ipynb`

3. **Скопируйте код из HTML:**
   - В браузере вы увидите все ячейки с кодом
   - Копируйте каждую ячейку и вставляйте в новый блокнот
   - Не забудьте копировать markdown ячейки тоже!

4. **Установите необходимые библиотеки:**
   ```bash
   pip install pandas numpy matplotlib seaborn scipy scikit-learn openpyxl ipyleaflet
   ```

5. **Выполните все ячейки:**
   - Cell → Run All

6. **Экспортируйте результаты:**
   - File → Download as → HTML (.html)
   - File → Download as → PDF via LaTeX (.pdf)

### Вариант C: Использование nbconvert (если установлен)

Попробуйте конвертировать HTML обратно в notebook:

```bash
# Установка nbconvert если нужно
pip install nbconvert

# Попытка конвертации (может не сработать)
jupyter nbconvert --to notebook lab3-6.html --output lab3-task6.ipynb
```

**Внимание:** Этот метод не всегда работает корректно.
