## 🚀 Инструкция по выполнению

### Шаг 1: Разархивируйте файл

#### Windows:
- Правый клик на `lab3-7.zip`
- Выбрать "Извлечь все..." или "Extract All..."
- Укажите папку для распаковки

#### macOS:
- Двойной клик по файлу
- Или в терминале:
```bash
unzip lab3-7.zip
```

#### Linux:
```bash
unzip lab3-7.zip
```

### Шаг 2: Изучите содержимое

После разархивирования вы должны увидеть:
- Возможно, HTML файл с заданием (аналогично task6)
- Или Jupyter Notebook файл (.ipynb)
- Или файлы данных для анализа

### Шаг 3: Выполните задание

В зависимости от того, что в архиве:

#### Если есть .ipynb файл:
1. Откройте его в Jupyter
2. Выполните все ячейки
3. Экспортируйте результаты

#### Если есть HTML файл:
1. Следуйте инструкциям из task6
2. Создайте новый блокнот
3. Скопируйте код

#### Если есть только данные:
1. Создайте новый блокнот `lab3-task7.ipynb`
2. Загрузите и проанализируйте данные
3. Создайте визуализации

### Шаг 4: Установите библиотеки

```bash
pip install pandas numpy matplotlib seaborn scipy scikit-learn openpyxl
```

### Шаг 5: Экспорт

После выполнения:
- File → Download as → HTML (.html)
- File → Download as → PDF via LaTeX (.pdf)
