# Инструкция по созданию скриншотов

## Для Задания 1

Вам нужно сделать 2 скриншота выполнения команд в терминале.

### Скриншот 1: AWK команды

1. Откройте терминал (Terminal в macOS/Linux, Command Prompt в Windows)
2. Перейдите в папку проекта:
   ```bash
   cd путь/к/lab3_variant5
   ```

3. Выполните команды:
   ```bash
   awk '{print $1, $2, $5}' data/list_students.txt
   awk '{print $0}' data/list_students.txt
   ```

4. Сделайте скриншот окна терминала
5. Сохраните как `awk_terminal.png` в эту папку `images/`

### Скриншот 2: sed команды

1. В том же терминале выполните:
   ```bash
   sed 's/Яшин/Кашин/' data/list_students.txt
   sed '/ЭКТ/d' data/list_students.txt
   ```

2. Сделайте скриншот
3. Сохраните как `sed_terminal.png` в папку `images/`

## Альтернатива

Если делать скриншоты неудобно, можно:
1. Пропустить этот шаг
2. В notebook задания 1 просто отметить, что скриншоты должны быть здесь
3. При защите лабораторной показать выполнение команд в реальном времени

## Формат скриншотов

- **Формат:** PNG или JPG
- **Разрешение:** любое читаемое
- **Имена файлов:**
  - `awk_terminal.png` - для AWK команд
  - `sed_terminal.png` - для sed команд
