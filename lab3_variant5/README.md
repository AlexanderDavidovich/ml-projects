### Задание 2: Основы языка AWK  
**Файл:** `src/task2/lab3-task2.ipynb`

Изучение расширенных возможностей AWK:
- Манипуляция полями
- Условная обработка
- Регулярные выражения
- Агрегатные функции

### Задание 3: Потоковый редактор sed
**Файл:** `src/task3/lab3-task3.ipynb`

Примеры работы с sed:
- Замена текста
- Удаление строк
- Фильтрация данных
- Работа с диапазонами

### Задание 4: Вариант 5 ⭐
**Файл:** `src/task4/lab3-task4.ipynb`

**Задача 1:** AWK-программа для чтения RFC файла и поиска строк с "Network Protocol"
- Загрузка файла с https://www.rfc-editor.org/rfc/rfc-ref.txt
- Поиск и подсчет строк

**Задача 2:** AWK-скрипт для расчета параметров параллелограмма
- Функция расчета площади (S = a × h)
- Функция расчета периметра (P = 2(a + b))
- Вызов из main()

**Задача 3:** sed-команда для замены пробела на 0
- Замена пробела в начале строки на 0, если за ним следует цифра
- Использование регулярных выражений и групп захвата

## Краткая инструкция по запуску

### 1. Клонирование репозитория
```bash
git clone <URL_РЕПОЗИТОРИЯ>
cd lab3_variant5
```

### 2. Установка зависимостей
```bash

conda create -n lab3 python=3.12

```
